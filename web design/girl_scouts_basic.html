Lists

You can also make lists! <ol> is an "ordered" list with numbers. <ul> is an "unordered" list with bullet points. Each line in the list is made with an <li> tag.

Images

You can add images to your website using the <img> tag. This tag is special because it doesn't have a closing </img> tag.

Links

Links are made with the <a> tag. The URL goes into the href property.

Divisions (sections)

The <div> tag is super useful for adding formats and dividing up your webpage. You can add styles that control the height, width, color, opacity (how much you can see through it) and much, much more! These are also fun for adding interactivity to your page.
