Today we learned about using HTML and CSS to make webpages!

There were 10 girls in today's class, and they each showed a lot of interest in creating a new web page!

HTML (HyperText Markup Language)

We learned about the basic structure of an HTML document, including the document type declaration and the head and body elements.

We learned about using tags to indicate different HTML elements, and the importance of always including the closing tag!

We talked about the following elements:

  • head
  • body
  • headings (h1, h2, h3, h4, h5, h6)
  • paragraphs (p)
  • images (img)

To learn more about HTML, check out the documentation here!

CSS (Cascading Style Sheets)

We wrote a few custom CSS rules to style our pages, adding custom colors to spice up our design.

Here are a few of the properties we used in our styles:

  • background-color
  • color
  • font-size

To learn more about CSS, check out the documentation here!

Typing Skills

We also learned to use some less-familiar characters, such as <, >, {, }, and ;.

Everyone used a few useful keyboard shortcuts (CTRL+C for "copy", CTRL+V for "paste" and CTRL+X for "cut").
