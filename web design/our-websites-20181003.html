HTML (HyperText Markup Language)

We learned about the basic structure of an HTML document, including the document type declaration and the head and body elements.

We learned about using tags to indicate different HTML elements, and the importance of always including the closing tag!

We talked about the following elements:

  • head
  • body
  • headings (h1, h2, h3, h4, h5, h6)
  • paragraphs (p)
  • images (img)
  • tables (table, thead, tbody)
  • line breaks (<br>)
  • horizontal dividers (<hr>)
  • divisions (div)

To learn more about HTML, check out the documentation here!

CSS (Cascading Style Sheets)

We wrote a few custom CSS rules to style our pages, adding custom colors to spice up our design.

Here are a few of the properties we used in our styles:

  • background-color
  • background
  • color
  • padding
  • margin
  • width

To learn more about CSS, check out the documentation here!

Keep on working!

On each page, the link to your HTML project is included in a link at the top. Copy your code into a file on your computer (for example in Notepad or Textedit) and save with a .html file extension. You can then open the page in your browser (Chrome, Safari, IE, Firefox, etc.)!

Pro Tip:

Have you ever seen a webpage and wondered about the HTML under the hood? Just right-click the page and select View page source to open the webpage's HTML in a new tab. This is a great way to explore advanced HTML and can even provide templates for your work. Just don't forget to cite your sources and check out copyright information!
